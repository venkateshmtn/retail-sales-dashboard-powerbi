 Project Overview

This project analyzes retail store sales data to uncover business insights such as revenue trends, profit performance, customer segments, and product category performance.

Tools Used-Power BI, Excel, DAX, Data Cleaning & Transformation

Dataset

The dataset contains: Order Date, Customer Segment, Region, Category & Sub-Category, Sales, Profit, Quantity

Key Insights:

Identified top-performing product categories

Analyzed monthly and yearly sales trends

Measured profit contribution by region

Built KPI metrics (Total Sales, Total Profit, Profit Margin)

Dashboard Features:

Interactive filters (Region, Category, Date)

KPI cards

Trend analysis

Profitability breakdown

Business Impact:

Helps management identify high-performing regions

Detects loss-making product categories

Improves decision-making using data-driven insights
